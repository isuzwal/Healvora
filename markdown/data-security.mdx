# Data Security Policy

_Last Updated: February 2026_

At **Healvora AI**, we are committed to ensuring the security and confidentiality of user data. As a healthcare-focused platform, we understand the importance of protecting sensitive information.

---

## 1. Data Protection Measures

We implement appropriate technical and organizational measures to protect user data, including:

- Secure data transmission using encryption
- Password-protected user accounts
- Role-based access control for administrators and doctors
- Secure cloud storage infrastructure

---

## 2. Access Control

Only authorized personnel are allowed to access sensitive data.  
Administrative access is restricted and monitored to prevent misuse.

---

## 3. Data Storage

All user information is stored securely on protected servers.  
We regularly monitor system activity to detect and prevent unauthorized access.

---

## 4. Data Breach Response

In the event of a data breach:

- The issue will be investigated immediately
- Affected users will be notified if necessary
- Corrective actions will be implemented to prevent recurrence

---

## 5. Continuous Improvement

We continuously improve our security practices to ensure data safety and system reliability.

---

If you have any questions regarding data security, please contact our support team.

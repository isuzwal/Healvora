# Privacy Policy

_Last updated: February 2026_

At **Healvora AI**, protecting user privacy is a core priority. This policy explains how we collect, use, and safeguard your data while delivering secure, AI-powered healthcare services.

## Information We Collect

- Personal details such as name, email, and contact information
- Health-related data shared for consultations and AI-based doctor matching
- Usage data to improve performance, reliability, and security

## How We Use Your Data

- Provide personalized medical services
- Improve AI recommendations and platform quality
- Maintain system security and prevent misuse

## Data Security

- End-to-end encryption
- Secure authentication and access controls
- Continuous monitoring and infrastructure hardening

## Data Sharing

We never sell user data. Information is shared only with trusted services when required.

## User Control & Rights

Users can access, update, or delete their data at any time.

## Data Retention

Data is retained only as long as necessary and securely deleted afterward.
